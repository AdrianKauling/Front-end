Trabalhando com Listas

Unodered Lists | <ul>

As <ul> são listas não ordenadas onde suas linhas não necessitam de uma ordem esécifíca.

Exemplo: Livros para ler

Definition List in practice

Definition List | <dl>
Cria uma lista com palavras e seus significados, semelhante ao dicionário

Definition Term | <dt>
É a palavra cuja a descrição a baixo explica o seu signicado

Definition description | <dd>
É a tag que contém o texto de significado do termo da tag <dt>
